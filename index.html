📖 Nemoz Label Generator 사용 방법

① 박스 번호가 존재하는 경우 (다품종 / 일반 납품서)

  1. Google 시트에서 납품서 탭의 전체 주소를 복사합니다.
  2. ① 구글 시트 URL 입력 칸에 붙여넣기 → 불러오기 클릭
  3. ② 라벨 공통 정보가 자동 채워집니다 (상품명 여러 줄도 OK)
  4. ③ 미리보기 확인 후 우측 상단 ⬇ PDF 다운로드
  5. PDF를 100mm × 100mm 실제 크기로 인쇄

② 공통 라벨인 경우 (No. of Boxes 컬럼 있는 경우)

  1. Google 시트에서 공통 라벨용 탭의 전체 주소를 복사합니다.
  2. ① 구글 시트 URL 입력 칸에 붙여넣기 → 불러오기 클릭
  3. ③ 미리보기에서 “共通ラベル”과 No. of Boxes 수만큼 라벨이 생성된 것을 확인
  4. ⬇ PDF 다운로드
  5. PDF를 100mm × 100mm 실제 크기로 인쇄
